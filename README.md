## 🚀 Getting Started

### Prerequisites

Make sure you have **Node.js** and **Git** installed.

```bash
node --version
git --version
```

### Installation

```bash
# Clone the repo
git clone https://github.com/Anushka0902/Portfolio.git

# Navigate into the project
cd Portfolio

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Visit **http://localhost:5173** to view it locally.
